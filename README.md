# 项目简介

## 核心功能
- 基于YOLOv8的水稻害虫检测系统，包含5229张标注数据集
- 支持6类水稻害虫检测：褐飞虱、绿叶蝉、稻纵卷叶螟、稻蝽、螟虫、稻蓟马
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整源码、训练好的模型、数据集和项目运行教程
- 界面功能包括：实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持选择显示标签和原图
- 提供批量检测功能，可查看所有检测结果
- 支持检测结果保存

## 适用场景
- 农业科研与害虫监测
- 水稻种植田块害虫识别
- 农业技术推广与教学
- 农业自动化管理系统

## 技术特点
- 使用PyCharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供UI界面源文件和图标资源
- 已训练好模型，配置环境后可直接使用
